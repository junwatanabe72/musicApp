## musicApp

---

対象アーティストの曲名を当てるアプリです。  
Apple Music API のサンプル音源を使用しています。  
お気に入りの曲があれば、iTunes よりご購入をお願いいたします。  
os によって、音楽が流れたり流れなかったり(特に ios)。。  
多分大丈夫です。。
