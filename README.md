## 不動産所有期間チェッカー

---

不動産の所有期間を確認するアプリです。  
不動産を売却した際、所有期間によって税率が異なります。  
所有期間の考え方がややこしいので、直感的にわかりやすく判定してくれるツールがほしいなと  
長年、考えていたので自分で作成しました笑  
取得した年をタップするだけの簡単設計となっています。
